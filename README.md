# 婚礼请柬制作指南

## 项目概述

这是一个专为微信分享设计的婚礼请柬H5页面，具有卡片效果、自动音乐播放、响应式设计和完整的邀请函功能。

## 文件结构

```
wedding_invitation.html    # 主请柬页面
11.mp3                    # 背景音乐
新郎.png                  # 新郎照片
新娘.png                  # 新娘照片
cover.png                 # 封面图片
README.md                 # 使用说明
```

## 使用步骤

### 步骤1：个性化修改

1. **修改婚礼信息**
   - 打开 `wedding_invitation.html` 文件
   - 修改婚礼日期、时间、地点等信息（在婚礼信息部分）
   - 更新联系方式

2. **替换图片**
   - 确保以下图片文件存在：
     - `cover.png` - 封面图片（建议尺寸：800x600）
     - `新郎.png` - 新郎照片（建议尺寸：400x600）
     - `新娘.png` - 新娘照片（建议尺寸：400x600）
   - 如需更换图片，保持相同文件名或修改HTML中的图片路径

3. **更换背景音乐**
   - 替换 `11.mp3` 文件为您喜欢的婚礼音乐
   - 支持MP3格式，建议时长2-3分钟

### 步骤2：微信分享优化

1. **修改分享信息**
   ```html
   <!-- 在HTML头部修改以下内容 -->
   <meta property="og:title" content="您的婚礼标题">
   <meta property="og:description" content="您的婚礼描述">
   <meta property="og:image" content="封面图片URL">
   ```

2. **获取分享链接**
   - 需要将HTML文件部署到服务器
   - 获得一个可公开访问的URL

### 步骤3：部署到服务器

#### 方案A：使用免费托管服务（推荐）
1. **GitHub Pages**
   ```bash
   # 1. 创建GitHub仓库
   # 2. 上传所有文件
   # 3. 启用GitHub Pages
   # 4. 访问 https://用户名.github.io/仓库名/wedding_invitation.html
   ```

2. **Vercel / Netlify**
   - 拖拽上传文件夹
   - 自动生成可访问链接

3. **腾讯云静态网站托管**
   - 适合国内访问
   - 速度快，支持HTTPS

#### 方案B：自有服务器
1. 上传文件到服务器
2. 通过域名访问
3. 配置HTTPS（微信要求）

### 步骤4：微信分享

1. **生成分享卡片**
   - 在微信中打开请柬链接
   - 点击右上角"..."菜单
   - 选择"发送给朋友"或"分享到朋友圈"

2. **分享文案建议**
   ```
   💒 诚挚邀请 💒
   
   我们即将步入婚姻殿堂
   诚邀您见证这一幸福时刻
   
   👉 点击查看电子请柬
   ```

## 常见问题

### Q1: 音乐无法自动播放？
A: 微信浏览器限制自动播放，需要用户交互。页面加载后点击任意位置即可播放。

### Q2: 分享卡片不显示图片？
A: 确保og:image的URL是绝对路径且可公开访问。

### Q3: 页面加载慢？
A: 优化图片大小，使用CDN，或选择国内服务器。

### Q4: 表单数据如何收集？
A: 需要后端支持。可以使用腾讯云开发、阿里云函数等Serverless服务。
